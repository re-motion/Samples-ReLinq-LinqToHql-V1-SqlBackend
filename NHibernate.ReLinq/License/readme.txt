NHibernate.ReLinq is an NHibernate (www.nhibernate.org) Linq-provider which is based on the 
generic Linq provider re-linq component from the open source re-motion framework (http://www.re-motion.org/).

For usage and currently supported Linq features please see "IntegrationTests.cs" 
in the "NHibernate.ReLinq.Test"-project.
